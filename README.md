# [한경×이랜드] K-Tour Local eXperience 콘텐츠 마케팅 과정

헤드카피: 푸른 제주 바다를 품고, 이야기는 콘텐츠가 된다
서브카피: 파도 소리가 영감(靈感)이 되는 곳, 3개월간 제주에서 로컬관광의 새로운 경험을 기획할 청년 마케터를 모집합니다

본 과정은 고용노동부가 주관하는 K뉴딜 아카데미 사업으로 한경닷컴이 운영하며 이랜드파크가 참여하는 숙박형 런케이션(Learning + Vacation) 교육 프로그램입니다. 제주 바다 앞에서 살며 생성형 AI 기술을 활용한 로컬관광 콘텐츠를 발굴하고 마케팅 실무 경험, 취업 지원 혜택을 받을 수 있습니다.

## 📌 주요 특징
* **3개월 숙박형 교육**: 제주 켄싱턴리조트 한림에서 동료들과 함께 거주하며 교육에 몰입합니다.
* **실무 중심 교육**: 콘텐츠 기획, 생성형 AI를 활용한 콘텐츠 제작, 온라인 광고 집행까지 마케팅 전 과정을 실전으로 배웁니다.
* **강력한 취업 혜택**: 수료생 전원 이랜드파크 서비스인턴십 지원 시 서류전형 면제 혜택을 제공합니다.
